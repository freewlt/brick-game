# 赢了个赢 🚗

> 微信小游戏 · 三消赢豪车，逐梦人生巅峰

一款以汽车为主题的三消闯关小游戏。点击棋盘上的车块放入槽中，集齐 3 辆同款即可赢得！用有限的步数清空棋盘，挑战 30 关递进难度，通关全部关卡解锁专属庆典彩蛋。每天还有一期限时高难的每日挑战等你征服。

---

## 玩法规则

| 操作 | 说明 |
|------|------|
| 点击车块 | 将顶层车块移入底部槽位 |
| 集齐 3 辆同款 | 自动消除并得分 |
| 槽位满（6格）且无法消除 | 本关失败 |
| 步数用光仍有残余车块 | 本关失败 |
| 清空棋盘 | 本关通关，获得 1~3 星 |

### 遮挡系统
同列**上方任意一行**有车时，该格被锁定（显示 🔒），无法点击。需先清除上方车块才能解锁。

### 槽位同色靠拢
新车入槽时，自动插入到同色车辆的紧邻位置，方便凑齐 3 辆消除。

### 道具系统
每关固定配发 **扩槽 ×1** 和 **洗牌 ×1**：
- **➕ 扩槽**：将槽位从 6 格扩展至 7 格，提供更多操作空间
- **🔀 洗牌**：随机重排棋盘上所有车块，打破僵局
- 道具用完后可通过**看广告**或**分享给好友**额外获取

### 撤销道具
每关提供 **1 次撤销**机会，可还原上一步操作，关键时刻救命用。

### 星级评定
通关后根据剩余步数比例评定 1~3 星：
- ⭐⭐⭐ 剩余步数 ≥ 50%
- ⭐⭐ 剩余步数 ≥ 30%
- ⭐ 剩余步数 < 30%

## 关卡设计

共 **30 关**，分三阶段递进：

### 手工精调（第 1~10 关）

| 关卡 | 车型数 | 堆叠层数 | 总车数 | 步数上限 | 难度 |
|------|--------|----------|--------|----------|------|
| 第 1 关 | 3 | 2 | 27 | 45 | ⭐ 入门 |
| 第 2 关 | 4 | 2 | 36 | 50 | ⭐ |
| 第 3 关 | 4 | 3 | 36 | 52 | ⭐⭐ 引入堆叠 |
| 第 4 关 | 5 | 2 | 45 | 52 | ⭐⭐ |
| 第 5 关 | 5 | 3 | 45 | 56 | ⭐⭐ |
| 第 6 关 | 5 | 3 | 60 | 75 | ⭐⭐⭐ |
| 第 7 关 | 6 | 3 | 54 | 68 | ⭐⭐⭐ |
| 第 8 关 | 6 | 3 | 72 | 90 | ⭐⭐⭐⭐ |
| 第 9 关 | 7 | 3 | 84 | 103 | ⭐⭐⭐⭐ |
| 第 10 关 | 8 | 3 | 96 | 116 | ⭐⭐⭐⭐⭐ 高难 |

### 算法生成（第 11~30 关）

难度公式：`maxMoves = ceil(carTypes × setCount × ratio)`，ratio 越低越难。

| 阶段 | 关卡 | 特点 | 步均容差 |
|------|------|------|----------|
| 进阶 | 11~16 | carTypes 5~7，layerMax 3，setCount 4~5 | 1.18~1.22 |
| 硬核 | 17~23 | carTypes 6~8，layerMax 4，setCount 5~6 | 1.16~1.18 |
| 地狱 | 24~30 | carTypes 8，layerMax 5，setCount 6~8 | 1.15 |

## 每日挑战模式

每天凌晨自动刷新一期固定关卡，所有玩家当日参数完全相同（用 `YYYYMMDD` 日期字符串作为 LCG 伪随机种子生成）。

| 特性 | 说明 |
|------|------|
| 每日一关 | 每天只能挑战 1 次，次日凌晨自动重置 |
| 高难度 | 车型 6~8 种，堆叠 3~5 层，步均容差 1.16~1.20 |
| 不消耗机会 | 挑战失败不扣除普通游戏机会 |
| 专属成就 | 解锁 3 个每日成就（首次通关 / 连续 3 天 / 连续 7 天） |
| 连续记录 | 记录每日连胜天数与累计通关次数，在挑战页面实时展示 |

- **好友排行榜**：通过微信云存储展示好友通关进度排名
- **机会系统**：每位玩家拥有 3 次游戏机会，失败消耗 1 次，30 分钟自动恢复 1 次
- **分享得机会**：分享给好友后可获得额外 1 次机会

## 技术实现

- **渲染**：微信小游戏 Canvas 2D API，纯 JS 手写渲染循环（`requestAnimationFrame`）
- **高清适配**：读取 `devicePixelRatio`，Canvas 物理像素 = 逻辑尺寸 × DPR，ctx 整体 scale 缩放
- **动态安全区**：`safeTop` 通过 `wx.getWindowInfo().statusBarHeight + 44` 动态计算，适配刘海屏、安卓各机型，不再硬编码 88px
- **场景管理**：StartScene / GameScene / ResultScene / AllClearScene / AchievementScene / LeaderboardScene / SettingsScene 七个场景类，通过 `Game.currentScene` 切换
- **音效系统**：基于 `wx.createWebAudioContext` 的 WebAudio 合成方案，无需音频文件，支持 BGM、消除音、连消音、胜负音、扩槽音等，音效开关状态持久化
- **社交**：`wx.setUserCloudStorage` 存储通关进度，`wx.getFriendCloudStorage` 获取好友数据

## 本地开发

1. 安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 导入项目目录（选择 `brick-game/` 文件夹）
3. 填写自己的 AppID（或使用测试号）
4. 点击编译预览即可运行

> 注意：好友排行榜、分享得机会等社交功能需要真机预览才能正常调用微信 API。

### 上线前需填写

| 位置 | 说明 |
|------|------|
| `src/utils/storage.js` → `SHARE_CONFIG.imageUrl` | 分享封面图地址（5:4 比例，建议 1000×800px CDN 链接） |
| `src/scenes/GameScene.js` → `_showRewardedAd()` | 替换 `'YOUR_AD_UNIT_ID'` 为真实微信流量主广告单元 ID |
